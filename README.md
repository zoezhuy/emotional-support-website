# Emotional Support Companion —— AI 情感支持对话产品
AI-Powered Emotional Support Web Application

---

**What is this? 这是什么**

一款基于 Next.js 构建的轻量级 AI 情感支持网站，通过对话交互、情绪分析与情绪趋势追踪，
探索 AI 在心理健康支持场景中的产品形态与 HCI 设计。

A calm, lightweight emotional support web app built with Next.js, exploring how AI can provide 
meaningful support through conversational interaction, emotion analysis, and trend tracking.

---

**Core Features 核心功能**

- 💬 对话页：支持消息输入、情绪标签识别与 AI 回复交互
- 📊 情绪分析页：分析用户情绪状态与对话模式
- 📈 情绪趋势页：追踪用户情绪变化趋势
- ⚠️ 所有页面包含免责声明：本工具仅供情感支持，不替代专业心理咨询

---

**Product Thinking 产品思路**

这个项目探索的核心问题是：**AI 如何在情感支持场景中建立信任感、提供有温度的交互体验？**

- 如何设计让用户感到安全、低压力的对话界面
- 情绪识别如何转化为可视化的用户洞察
- AI 回复的边界意识——什么时候该引导用户寻求专业帮助
